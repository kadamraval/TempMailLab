# **App Name**: TempInbox

## Core Features:

- Generate Temp Email: Generate random temporary email
- Copy to Clipboard: Copy-to-clipboard button for email
- Inbox View: Inbox view showing sender name, subject line, and time received
- Auto-Refresh Inbox: Auto-refresh inbox every few seconds
- View Email Content: View full email content (HTML + plain text)
- Expiration Timer: Email expiration countdown timer
- Generate New Email: Generate new email on demand
- Responsive UI: Responsive UI (mobile & desktop)
- AdSense - Homepage: Display ad on homepage (below generated email)
- AdSense - Inbox: Display ad between inbox messages
- AdSense - Email Viewer: Display ad in email viewer
- AdSense - Sticky Footer: Sticky footer ad for mobile
- AdSense - Privacy Policy: Cookie/Privacy policy for AdSense approval
- User Login: Google/email login (optional)
- Save Inbox History: Save inbox history (for logged-in users)
- Ad-Free Premium: Ad-free experience for premium users
- Longer Expiration: Longer email expiration time (30 min, 1 hr, 1 day)
- Custom Prefix: Custom email prefix (e.g., yourname@mail.tm)
- Email Forwarding: Email forwarding to real email address
- Inbox Search/Filter: Inbox search/filter
- Export Inbox: Export inbox (ZIP or PDF)
- Admin - Dashboard: Admin dashboard to view stats: Total users, Total inboxes generated, Emails received
- Admin - Delete Abuse: Delete abuse inboxes
- Admin - Block IPs/Domains: Block IPs or sender domains
- Admin - User Sessions: View active user sessions
- Admin - AdSense Tracking: Track AdSense performance (external)
- Dev - API Access: Developer API access (create inbox, fetch email)
- Dev - API Key Gen: API key generation
- Dev - Usage Logs: Usage logs per developer
- Dev - Paid API Plans: Paid API plans for higher limits

## Style Guidelines:

- Primary color: A calming, muted blue (#64B5F6) to create a sense of trust and reliability.
- Background color: A light gray (#F5F5F5) for a clean and neutral backdrop.
- Accent color: A vibrant green (#4CAF50) to highlight important actions like copying the email address or viewing an email.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple, clean, and recognizable icons for actions like copy, refresh, and delete.
- Maintain a clear, intuitive layout with a prominent display of the generated email and inbox messages. Use spacing to improve readability.
